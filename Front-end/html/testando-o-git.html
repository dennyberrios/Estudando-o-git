Hellow world.

Configuração usuario no git.

git config --global user.name "seu nome"

git config --global user.email "seu email de cadastro do github"

git config --list

Upa projetos no github.

git init

git add .

git commit -m "nome do projeto"

git branch -m main

git remote add origin (link do github)

git push -u origin main


Clona conteúdo do github.

Cria uma copia exata no computador de todos os arquivos
existentes em um repositorio remoto.

git clone ( link do repositorio )

Baixa apenas as atualizações.

git pull
